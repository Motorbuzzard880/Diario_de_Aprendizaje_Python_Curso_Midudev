# 🔄 *
**"*": Puede aparecer 0 o más veces**
```python
text = "aaaba"
pattern = "a*"
matches = re.findall(pattern, text)
print(matches)
```

# 🔄 +
**+: Una a más veces**
```python
text = "dddd aaa ccc a bb aa casa"
pattern = "a+"
matches = re.findall(pattern, text)
print(matches)
```

# 🔄 ?
**?: Cero o una vez**
```python
text = "aaabacb"
pattern = "a?b"
matches = re.findall(pattern, text)
print(matches)
```

# 🔄 {n}
**{n}:  Exactamente n veces**
```python
text = "aaaaaa         aa   aaaa"
pattern = "a{3}"
matches = re.findall(pattern, text)
```
